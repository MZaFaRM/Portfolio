Healie

: Personal Health Assistant :

  Comes from the words "Health" and "Genie", Healie is a personal health assistant that we built for a Hackathon, the project actually went on to win "Best AI/ML project" in the competition. It's supposed to work like a health assistant that you can upload your health diagnosis and test data into, from where it would provide a secondary opinion regarding your treatments, we thought such a project would be important because medical misdiagnosis happen all the time.

  Tldr; The project helps doctors in making better choices and patients in being better educated of their treatments and medicines.

DataForge

: Database Auto-Population tool :

 When I was doing the tedious task of populating databases for testing an API I wrote, I thought, "There's got to be a better way!" And that's how DataForge came to life.

 It's a Python tool I built to auto-generate and insert realistic test data into databases. What's cool about it is how it figures out those foreign key relations to order those tables to fill first. I've had a great time making data insertion a breeze with this project! Hope you find it helpful too.

Skills: Python | SQLAlchemy
