DataForge

: Database Auto-Population tool :

 When I was doing the tedious task of populating databases for testing an API I wrote, I thought, "There's got to be a better way!" And that's how DataForge came to life.

 It's a Python tool I built to auto-generate and insert realistic test data into databases. What's cool about it is how it figures out those foreign key relations to order those tables to fill first. I've had a great time making data insertion a breeze with this project! Hope you find it helpful too.
